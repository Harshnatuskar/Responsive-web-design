intro

C++ is a cross-platform language that can be used to create high-performance applications.

  • What is C++:

    • Cross-platform language that can be used to create high-performance applications
    • Extension to the C language
    • Gives programmers a high level of control over system resources and memory

    Why is it used:

    • Worlds most popular programming language
    • Can be found in operating systems, GUIs and embedded systems
    • It is object oriented language, which gives a clear structure to the programs
    • Portable

It is similar to C but supports classes and objects , when C does not.

syntax
  • Here is some C++ code:
    • #include using namespace std; int main() { cout << "Hello World!"; return 0; }

You need to install GCC to compile the code.

An IDE like VS-Code is also preferred for coding

output

The cout object, together with the << operator, is used to output values/print text:

  • You can add as many cout objects as you want. However, note that it does not insert a new line at the end of the output:
    • #include using namespace std; int main() { cout << "Hello World!"; cout << "I am learning C++"; return 0; }
comment

Comments can be used to explain C++ code, and to make it more readable.

  • There are two kinds of comments:

    • single line comment
    • multiline comment
    • // This is a single line comment /* This is a multi-line comment*/
variable

Variables are containers for storing data values

  • For example:

    • int: store integers
    • double: stores floating point numbers
    • char: stores single characters
    • string: stores text
    • bool: stoes values with two states: true / false
    int num = 5; double float = 5.99; char letter = 'D'; string text = "Hello"; bool value = true;
